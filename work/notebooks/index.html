Content Refresh Opportunity Playbook: Search Intelligence Model

Author: Muhammad Shahmeer

Abstract

This study addresses content performance decay by engineering an automated machine learning model to identify search pages requiring updates. Using DuckDB to process over 7.8 million search daily performance records hosted on Hugging Face, we engineered month-over-month traffic trend features. Pages experiencing a 20% or greater traffic decline were labeled as targets for content refresh. A Random Forest model evaluated using a time-aware split achieved a strong 0.9482 ROC AUC score without data leakage. The resulting ranked action engine automatically prioritizes at-risk pages with clear diagnostic reason codes for SEO workflows.

1. Introduction & Problem Statement

Organic search traffic decays over time due to changing user search intent, competitive content, and search engine algorithm updates. Manually auditing thousands of URLs to determine which pages need content updates is inefficient. This project builds a machine learning pipeline that identifies declining content and prioritizes pages for refresh actions.

2. Data & Data Leakage Prevention

The analysis utilizes the FlyRank ML Internship dataset hosted on Hugging Face, containing approximately 7,883,565 aggregated search performance rows. Features were calculated by aggregating daily performance into monthly metrics (gsc_clicks, gsc_impressions). To strictly prevent data leakage, training data was partitioned strictly prior to the test window (June 2026), ensuring future performance data was not accessible during model training.

3. Methodology

A target variable (target_needs_refresh) was defined as 1 if a page's clicks dropped by 20% or more month-over-month, and 0 otherwise. A Random Forest Classifier was trained using past performance features (feature_clicks, feature_impressions). Predictions were mapped to probability scores to rank pages by decay risk.

4. Results & Performance

The baseline model achieved strong predictive discrimination across time-separated test data:

  • ROC AUC Score: 0.9482
  • Overall Accuracy: 91%
  • Class 1 Precision: 0.63
  • Class 1 Recall: 0.55

6. Limitations & Honest Framing

The predictions provide directional support for content teams rather than causal guarantees. External factors such as seasonality and broader algorithm updates are not explicitly modeled in this feature set.
